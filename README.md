# Learnify – Online Learning Platform (Capstone Project)

## Project Title & Theme
**Project Title:** Learnify  
**Theme:** Online Learning Platform  

Learnify is a responsive static website designed as an online learning platform where users can explore courses, instructors, and educational resources through a clean and modern interface.

---

## Short Description of Sections Implemented

- **Header & Navigation:**  
  Includes the platform name, course search bar, and Login/Sign Up buttons.

- **Hero Section:**  
  A banner section with a main heading, short tagline, and call-to-action for searching courses.

- **Popular Categories:**  
  Displays learning categories such as Development, Design, Business, Languages, and more using grid-based cards.

- **Featured Courses:**  
  Course cards showing images, titles, ratings, and pricing with visual filter options.

- **Instructors Section:**  
  Profiles of instructors with images, roles, and short descriptions.

- **Testimonials Section:**  
  Student feedback cards to showcase user experience and trust.

- **Blog Section:**  
  Educational articles displayed in a responsive card layout.

- **FAQ Section:**  
  Frequently asked questions implemented using HTML `<details>` and `<summary>` tags.

- **Newsletter Section:**  
  Email subscription form for updates and announcements.

- **Footer:**  
  Contains quick links, contact details, and social media icons.

---

## Breakpoints Used & Layout Techniques

### Layout Techniques
- **HTML5 Semantic Elements** for structured and accessible markup  
- **CSS Flexbox** for alignment and navigation layout  
- **CSS Grid** for course cards, categories, blogs, and instructor sections  

### Responsive Breakpoints
- **Desktop:** ≥ 1024px  
- **Tablet:** ≤ 900px  
- **Mobile:** ≤ 600px  

Media queries are used to stack layouts, adjust font sizes, and convert multi-column grids into single-column layouts on smaller screens.
